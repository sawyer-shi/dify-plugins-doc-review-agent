## English Version

### Overview

The Document Review Agent Plugin ("Plugin") is designed to provide AI-powered document review capabilities for various types of documents including tender documents, official documents, contracts, and materials. This privacy policy explains how we handle your data when you use our plugin within the Dify platform.

### License Information

This plugin is licensed under Apache License 2.0, which is a permissive open-source license that allows for use, modification, and distribution of software. The full license text is available in the LICENSE file included with this plugin.

### Data Collection

**What we DO NOT collect:**
- Personal identification information
- User account details
- Usage analytics or tracking data
- Device information
- Location data
- Cookies or similar tracking technologies

**What we process:**
- Document files for review (docx format)
- Text prompts for document parsing and analysis
- Review rules and audit criteria
- Document chunks for risk assessment
- Audit reports and risk summaries
- Model configuration parameters

### Data Processing

**LLM Processing:**
- All document parsing and review requests are sent to the configured LLM model
- Document content is processed in chunks for comprehensive analysis
- Review rules and criteria are used to guide the AI analysis
- Risk assessment is performed based on the loaded rules
- Document annotation is generated using LLM assistance

**Local Processing:**
- Plugin configuration and settings are stored locally
- Document files are parsed and sliced locally before LLM processing
- Temporary files may be created during document processing
- No permanent data storage beyond plugin session

### Data Storage

**No Local Data Retention:**
- We do not store copies of your uploaded documents after processing
- We do not retain audit reports or risk summaries locally
- We do not store generated annotations beyond the current session
- Model configuration is stored securely in Dify's plugin configuration

**LLM Data Handling:**
- LLM providers process your document content according to their privacy policies
- Generated content is processed and returned in real-time
- Users should review their LLM provider's privacy policy for data handling practices

### Security Measures

**Data Protection:**
- All communication with LLM APIs uses encrypted connections
- Plugin operates within Dify's secure environment
- No unauthorized network connections are made
- Document files are processed securely

**Third-Party Services:**
- This plugin relies on LLM services configured in Dify
- Users must have valid LLM model access
- All AI processing occurs on the configured LLM infrastructure

## 中文版本

### 概述

文档审核Agent插件（"插件"）旨在提供AI驱动的文档审核功能，支持标书、公文、合同、资料等各种类型的文档审核。本隐私政策说明了您在 Dify 平台内使用我们的插件时，我们如何处理您的数据。

### 许可证信息

本插件采用 Apache License 2.0 许可证，这是一个宽松的开源许可证，允许使用、修改和分发软件。完整的许可证文本可在本插件附带的 LICENSE 文件中找到。

### 数据收集

**我们不收集的内容：**
- 个人身份信息
- 用户账户详细信息
- 使用分析或跟踪数据
- 设备信息
- 位置数据
- Cookie 或类似跟踪技术

**我们处理的内容：**
- 用于审核的文档文件（docx 格式）
- 用于文档解析和分析的文本提示词
- 审核规则和审核标准
- 用于风险评估的文档切片
- 审核报告和风险摘要
- 模型配置参数

### 数据处理

**LLM 处理：**
- 所有文档解析和审核请求都发送到配置的 LLM 模型
- 文档内容以切片形式进行综合分析
- 审核规则和标准用于指导 AI 分析
- 基于加载的规则执行风险评估
- 使用 LLM 辅助生成文档批注

**本地处理：**
- 插件配置和设置存储在本地
- 文档文件在 LLM 处理前进行本地解析和切片
- 文档处理期间可能会创建临时文件
- 除插件会话外，不进行永久性数据存储

### 数据存储

**不保留本地数据：**
- 处理后我们不存储上传文档的副本
- 我们不在本地保留审核报告或风险摘要
- 我们不存储当前会话之外的生成批注
- 模型配置安全存储在 Dify 的插件配置中

**LLM 数据处理：**
- LLM 提供商根据其隐私政策处理您的文档内容
- 生成的内容实时处理并返回
- 用户应查看其 LLM 提供商的隐私政策了解数据处理做法

### 安全措施

**数据保护：**
- 与 LLM API 的所有通信使用加密连接
- 插件在 Dify 的安全环境中运行
- 不进行未经授权的网络连接
- 文档文件安全处理

**第三方服务：**
- 本插件依赖 Dify 中配置的 LLM 服务
- 用户必须拥有有效的 LLM 模型访问权限
- 所有 AI 处理都在配置的 LLM 基础设施上进行
